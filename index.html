What's New v27.7.3A
Released 23 March 2026 · v27.7.3A · Patch Release
📈
01/08
Fix
Predictive Pacing Fixed
position:relative was missing from the bar — marker was positioning relative to a distant ancestor. Also switched to due-only formula and added colour-coded markers (green / gold / red).
🏷️
02/08
Fix
Drill List Label Bug Fixed
All topics from the same day shared an identical label (e.g. "National & International" × 7). Now each topic shows its number within the session: "National & International · 2/7".
🔄
03/08
Feature
Drill List Refresh Integrated
The "Refresh Intelligence" toggle in the profile dropdown now also re-renders the Drill List. The standalone Refresh button on the section was removed — one button refreshes everything.
📋
04/08
Layout
Drill List Layout Improved
3-column grid replaced with a single-column accordion. Each subject card spans full width, giving the expandable topic list proper space to breathe.
📅
05/08
Feature
262 Special Days
SpecialDays expanded from 13 to 262 entries — every Indian national day, UN/WHO observance, armed forces day, Kerala day, and international event. Exam day (May 10) marked specially.
🛠️
06/08
Fix
Drill List Same-Content Fix
render() was returning cached state — same subjects showed even after topics were marked done. Now reads fresh from localStorage on every call; _expanded resets on force refresh.
✅
07/08
Stable
Stable Release
v27.7.3A is a stable patch on top of v27.7.3. All known Drill List and Predictive Pacing bugs resolved. Marked STABLE — suitable for daily use through exam day.
📦
08/08
Build
Build · v27.7.3A
Released 23 March 2026. Patch release on v27.7.3 — pacing fix, drill fixes, 262 special days, layout polish. File: ~299 KB.
